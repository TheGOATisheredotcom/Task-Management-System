# Python CLI Task Manager

A robust Command Line Interface (CLI) application to manage daily tasks, featuring data persistence, priority levels, and CSV export capabilities.

## üöÄ Features
- **Full CRUD:** Add, view, and mark tasks as complete.
- **Organization:** Categorize tasks (Work, Personal, Urgent, etc.).
- **Prioritization:** Assign Low, Medium, or High priority to tasks.
- **Smart Filtering:** View tasks by category, due date, or priority level.
- **Data Persistence:** Automatically saves tasks to a `tasks.json` file.
- **Export:** Save your task list to a `tasks.csv` file for use in Excel or Google Sheets.
